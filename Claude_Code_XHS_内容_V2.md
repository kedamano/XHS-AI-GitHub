## 小红书正文（优化版）

---

最近在写代码的时候，发现一个超好用的工具必须分享给大家。

就是 Anthropic 出品的 **Claude Code**。

之前一直在用 Cursor、Windsurf 这些工具，但 Claude Code 用下来真的有点不一样。

**先说说它解决什么问题：**

平时写代码，总有些重复性的工作要做——找某个函数在哪、解释一段老代码、帮我写个 Git 提交信息。这些事情不大，但打断思路真的很烦。

Claude Code 直接在终端里运行，说句话就能帮你搞定，不用切来切去。

**用了一段时间感受最深的几点：**

第一，它真的能读懂你的代码库。不是那种泛泛而谈的回答，是真的结合你项目的上下文来理解。

第二，常见的开发任务它都能帮上忙。读写文件、跑命令、处理 Git 操作，说句话就行。

第三，门槛不高。装好之后登录账号就能用，文档也写得挺清楚的。

---

**什么人适合试试：**

如果你平时写代码比较多，特别是需要经常处理不熟悉的代码库，或者想让一些重复性工作自动化，那可以体验一下。

免费额度基本够日常使用，上了班的基本不用担心不够用。

---

用了几个月下来，代码写的过程中确实少了一些打断，多了一点专注。

你们有什么提升效率的好工具吗？评论区聊聊看。
