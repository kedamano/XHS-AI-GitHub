# XHS-AI-GitHub 每日创作助手

> AI 时代的内容创作神器！将 GitHub 上的优质 AI 项目转化为小红书爆款种草内容

## 🎯 项目简介

这是一个专注于 **AI 内容创作** 的工具集，帮助你：

- 📊 从 GitHub 趋势中筛选优质 AI 项目
- ✍️ 将技术项目转化为小红书风格的种草内容
- 🔥 生成具有高传播力的标题、封面文案和正文
- #️⃣ 自动生成热门标签

## 📦 Skills 列表

### xhs-ai-blogger - 小红书 AI 博主

将 GitHub AI 项目转化为小红书爆款种草内容的工作流技能。

**功能特点：**
- 🎯 智能筛选适合推荐的 AI 项目
- 📝 生成 5 种类型爆款标题（情绪型、数字型、对比型、FOMO型）
- 📣 打造有冲击力的封面文案
- ✨ 完整正文模板（钩子→功能→理由→人群→评价→互动）
- #️⃣ 10-15 个热门标签组合
- ✅ 质量检查清单确保内容品质

**使用方式：**

在 WorkBuddy 中直接说：
```
帮我写一篇小红书推荐 XX 项目
生成 AI 项目的种草内容
写一个 AI 工具分享笔记
```

## 🚀 快速开始

### 安装 Skill

1. 克隆本仓库：
```bash
git clone https://github.com/kedamano/XHS-AI-GitHub.git
```

2. 将 Skill 安装到 WorkBuddy：
   - 将 `skills/xhs-ai-blogger/SKILL.md` 文件复制到 `~/.workbuddy/skills/xhs-ai-blogger/SKILL.md`

3. 重启 WorkBuddy 即可使用
