# OASYS 项目概览

## 项目简介

OASYS（Office Automation System）是一个基于 Spring Boot 的智能 OA 办公自动化系统，集成智谱AI大模型，提供全面的办公管理功能。

**GitHub**: https://github.com/kedamano/oasys

## 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 后端框架 | Spring Boot | 1.5.6.RELEASE |
| 数据持久 | Spring Data JPA + MyBatis | 1.3.0 |
| 数据库 | MySQL | 8.0 |
| AI能力 | 智谱AI GLM-4 | SDK 2.0.2 |
| 前端模板 | FreeMarker | - |
| UI框架 | Bootstrap | 3.3.7 |
| JS库 | jQuery | 1.11.3 |

## 核心亮点

1. **AI智能助手** - 右下角悬浮按钮，接入智谱GLM-4大模型
2. **全场景覆盖** - 10大功能模块覆盖企业办公全流程
3. **灵活流程引擎** - 多模板审批流（请假/报销/出差/转正/离职）
4. **现代化UI** - Bootstrap响应式设计，支持多端访问
5. **完善权限体系** - 角色-用户-权限三级管理
