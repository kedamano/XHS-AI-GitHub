# OASYS 核心模块详解

## 功能模块一览

### 核心办公模块

| 模块 | 功能描述 | 关键接口 |
|------|---------|---------|
| 用户管理 | 个人信息/密码修改/登录日志 | `/user/*` |
| 考勤管理 | 签到签退/考勤统计/请假加班 | `/attendce/*` |
| 流程管理 | 费用报销/出差/转正/离职 | `/process/*` |
| 任务管理 | 任务分配/跟踪/统计/进度监控 | `/task/*` |
| 邮件管理 | 邮件收发/模板/分类管理 | `/mail/*` |
| 日程管理 | 日程安排/提醒/日历视图 | `/daymanage/*` |
| 通讯录 | 内部/外部联系人/分组管理 | `/address/*` |
| 文件管理 | 上传下载/共享/版本管理 | `/file/*` |
| 讨论区 | 内部交流/话题讨论/投票 | `/chat/*` |
| 公告通知 | 系统公告/个人通知/消息提醒 | `/note/*` |

### 系统管理模块

- **角色权限管理** - 灵活的角色配置和细粒度权限分配
- **菜单管理** - 动态菜单配置，前端路由可控
- **系统监控** - 用户日志、操作记录、数据统计
- **审批流配置** - 可视化审批节点配置

### AI助手模块

OASYS集成了智谱AI GLM-4大模型，提供智能问答能力：
- 解答OA系统使用问题
- 协助发起各类申请流程
- 提供日程管理建议
- 快速定位系统功能
- 协助撰写邮件/公告
