# OASYS 小红书配图生成器

基于 [OASYS](https://github.com/kedamano/oasys) 智能 OA 办公自动化系统的**科技简约深色风**小红书配图生成器。

## 特性

- 科技简约深色风（6套主题色：蓝黑/紫黑/绿黑/橙黑/青黑/玫红）
- 零 emoji，纯文字排版，Windows 字体完美兼容
- 支持单图自定义 + 6张批量生成
- 1080x1440 小红书标准竖图

## 快速开始

```bash
cd scripts
pip install Pillow
python batch_oasys.py
```
